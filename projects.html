Face Emotion Recognition

Objective:

  • To identify various emotions projected on human faces to improve interaction between intelligent systems and human beings.
  • To analyze human facial expressions and detect six basic universal emotions: happiness, sadness, anger, fear, surprise, and disgust.
  • To enhance human‚Äìartificial interaction, allowing better control over AI systems using facial gestures and expressions.
Introduction:

Image Processing is a vast area of research with widespread applications. Facial expressions are non-verbal cues that reflect human emotions. Automatic facial expression recognition plays a significant role in artificial intelligence, robotics, personal identification, access control, videophone, and teleconferencing.

The main aim of this project is to develop an Automatic Facial Expression Recognition System that takes human facial images with expressions as input and classifies them into six different expression classes.

Existing Method:

Automatic Facial Expression Recognition has seen major development in recent years, with various notable researches published. For facial expression detection, developers have implemented different extraction methods like Linear Vector Quantization (LVQ), Supervised Descent Method (SDM), and Local Binary Pattern (LBP).

These conventional extraction techniques are often extremely complex and lead to lower recognition accuracy. Furthermore, many of these techniques are limited in recognizing only a few emotions.

Proposed Method:
  • The paper proposes Facial Emotion Recognition of a person through three steps:
    • Captured Image
    • Face Detection
    • Using Histogram of Oriented Gradient Feature
  • Histogram Of Gradient (HOG) is a window-supported feature descriptor that uses the gradient filter. The extracted features are based on the edge information of the registered face images. It extracts the visual features‚Äîfor example, a smile expression means curvature-shaped eyes.
Implementation of the HOG descriptor algorithm is as follows:
  • Divide the image into small connected regions called cells, and for each cell compute a histogram of gradient directions or edge orientations for the pixels within the cell. Discretize each cell into angular bins according to the gradient orientation. Each cell's pixel contributes weighted gradient to its corresponding angular bin. Groups of adjacent cells are considered as spatial regions called blocks. The grouping of cells into a block is the basis for grouping and normalization of histograms. Normalized group of histograms represents the block histogram. The set of these block histograms represents the descriptor.
Software/Hardware specifications:
  • MatLab (R2019b)
  • Machine Learning Algorithms
Conclusion:

The proposed facial emotion recognition system provides an efficient way to detect and classify human emotions using image processing techniques. By leveraging HOG feature extraction and classification through machine learning, the system is capable of recognizing key facial expressions with improved accuracy and reduced complexity compared to traditional methods. This enhances the interaction between humans and machines, making intelligent systems more responsive and human-centric.
